## start:
Running at linux system, you can use the command above to start the process. Make sure to replace `[input_picture_path]` with the path to your input image, `[number of color]` with the desired number of colors for the output image, and `[output_file.png]` with the desired name for your output file.

```
/nob_build -i [input_picture_path] -c [number of color] -o [output_file.png]
```
